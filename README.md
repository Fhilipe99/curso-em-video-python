## ▶️ Como executar

**1. Clone o repositório**
```bash
git clone https://github.com/Fhilipe99/curso-em-video-python.git
```

**2. Acesse a pasta do módulo desejado**
```bash
cd ".modulo 5"
```

**3. Execute o arquivo Python**
```bash
python aula10.py
```
